Gamri is light weight hyrpland rice just for playing games or if u don't want any excess features.
It has keybinds for Discord, Steam, Firefox and Thunar files.
U can share screen via Discord take screenshot (full, region).
It doesn't have any lock screen (sry i want to keep it minimal).

Step to install :

1. Clone repo :   
    git clone https://github.com/Paanyaa/Gamri.git ~/.gaming/
    cd ~/.gaming
    chmod +x setup.sh
    ./setup.sh
    
-- This will create a new session entry point called Hyprland-gaming to load my rice.
